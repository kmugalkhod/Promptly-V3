APP_NAME: portfolio-website
DESCRIPTION: Personal portfolio showcasing projects, skills, and contact information with smooth scrolling and responsive design.

DESIGN_STYLE: minimal
- Clean whites with subtle borders
- Ample whitespace for elegance
- Typography-focused layout
- Professional, uncluttered aesthetic

PACKAGES: none
- Default stack sufficient for portfolio needs

ROUTES:
- / (Hero section + navigation anchor)
- /#about (About section, background, skills)
- /#projects (Project showcase grid/list)
- /#contact (Contact form, links, CTA)

COMPONENTS:
- Navbar: Fixed/sticky header with logo and nav links
- HeroSection: Full-height intro with headline and CTA
- AboutSection: Bio, skills, tech stack showcase
- ProjectCard: Individual project with image, description, tech tags, link
- ProjectsGrid: Responsive grid/list of projects
- SkillBadge: Technology/skill pill badge
- ContactForm: Name, email, message fields using shadcn Form
- Footer: Copyright, social links, back-to-top
